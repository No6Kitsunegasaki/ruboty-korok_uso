# Ruboty::KorokUso

```
めかくしをしたひとのよこでぽたぽたやきをやくじっけんがあった。
```

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-korok_uso'
```

## Usage

```
@ruboty うそ
```
